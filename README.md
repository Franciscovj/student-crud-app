# Student CRUD App

## Overview
Student CRUD App é uma aplicação simples em PHP para gerenciar registros de alunos, permitindo criar, ler, atualizar e excluir informações de alunos.

## Tecnologias Utilizadas
- PHP
- MySQL
- HTML/CSS
- Bootstrap
